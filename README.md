[FAPSM](./FAPSM/README.md)
@shakibFrequencyAdaptivePhase2017
利用变频+外移相调制的方法实现LLC谐振变换器的宽范围电压范围输入。
等效SRC，但利用励磁电感间接实现了次级侧开关管的ZVS导通。但关断电流较大。
适用于宽范围且增益接近于1的High Voltage to High Voltage 变换器。
包含参数设计文件、仿真文件。
